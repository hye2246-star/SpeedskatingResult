# 2026 전국남녀 스피드스케이팅대회 — 조주원 경기 기록

일자: 2026-03-28
대회명: 2026 전국남녀 초/중/고/대/실업 스피드스케이팅대회
소속: 강명중

## 매스스타트 (중/고등 여자) — 최종 3위 🥉

### 경기 운영
- 10바퀴, 포인트 배분: 4바퀴(20/10/5), 7바퀴(20/10/5), 결승선(40/20/10)
- 8바퀴까지 손하나(3학년, 최강자) 뒤에서 체력 비축
- 2바퀴 남기고 스퍼트 → 기록상 2위 피니시
- 포인트 합산 최종 3위

### 주요 선수 분석
| 선수 | 학년 | 전략 | 결과 |
|------|------|------|------|
| 오정선 | 1학년 | 뒷그룹에서 중반 이후 독주 | 1위 |
| 한마음 | - | 초반 단독 선두, 포인트 챙김 | 2위 (포인트) |
| 조주원 | 1학년 | 8바퀴 체력 비축 → 막판 스퍼트 | 3위 |

### 3학년 우승후보: 손하나, 전서현, 최운서
### 2학년 우승후보: 안경인, 최하연, 박서영
→ 언니들 서로 눈치싸움 → 동생들(오정선, 주원이)이 치고 나감

### 평가
- 침착함, 집중력, 인내가 강점
- 순간 판단력 뛰어남
- "똑똑한 경기" — 머리로 이긴 경기

## 500m — 1학년 1위 🥇

| 항목 | 결과 |
|------|------|
| 전체 순위 | 18명 중 8위 |
| 1학년 순위 | 1위 |
| 기록 | 종전 대비 1초 단축 |

## 1000m — 중1 압도적 1위 🥇

| 순위 | 선수 | 기록 |
|------|------|------|
| 1위 | 김하엘 | 1:25.18 |
| 2위 | 안경인 (2학년) | 1:25.56 |
| 3위 | 차윤지 (같은팀) | 1:25.98 |
| 4위 | 최운서 (3학년) | 1:26.47 |
| 5위 | 한마음 (3학년) | 1:26.58 |
| - | 조주원 (중1) | 1:29.25 |

→ 상위 전원 2~3학년, 중1로서 3~4초 차이는 체격 차이 (정상)
→ 같은 중1 중 압도적 1위

## 라이벌: 차윤지 (같은 팀, 별가람중)

| 항목 | 조주원 | 차윤지 |
|------|--------|--------|
| 체격 | 작음 | +10cm, +10kg |
| 기술/자세 | 더 좋음 | 보통 |
| 힘/덩치 | 한계 | 강점 |
| 두뇌/전략 | 탁월 | 이번주 멍청한 플레이 |
| 매스스타트 | 우위 | 열세 |
| 장거리/스프린트 | 현재 열세 | 우위 |

→ 체격 차이 반만 줄어도 역전 가능 (엄마 분석)
→ 500m 베스트 기록은 주원이가 근소하게 앞섬

## 지난주 vs 이번주

| | 지난주 | 이번주 |
|--|--------|--------|
| 주원 | 중학 언니들 텃세에 5포인트·6위 | 매스 3위 + 500 1학년 1위 + 1000 중1 1위 |
| 윤지 | 1500·3000 각 4~5초 단축, 매스 2위 | 매스 해보지도 못함 |

→ 지난주 경험(기차놀이 텃세)이 이번주 바로 적용됨

## 성장 분석 및 추천

### 강점
- 기술/자세 이미 언니들급
- 두뇌/전략은 오히려 언니들보다 위
- 같은 학년 중 독보적

### 추천 방향
1. 매스스타트 집중 투자 — 두뇌+기술로 피지컬 커버 가능한 종목
2. 500m 단거리 갈고닦기 — 기술로 커버 가능, 기록 단축 중
3. 1000m 이상 장거리는 조급하지 않게 — 체격이 따라올 때까지
4. 윤지 비교 스트레스 관리 — 종목별로 분리해서 보기
